うるう年の判定

規則にのっとってわかりやすく書くとこのパターンになる

int IsLeapYear(int year)
{
    /* 4で割り切れる年はうるう年 */
    if (!(year % 4)) {

        /* ただし100で割り切れる年は平年 */
        if (!(year % 100)) {

            /* ただし400で割り切れる年はうるう年 */
            if (!(year % 400)) {
                return 1;
            }

            return 0;
        }

        return 1;
    }

    return 0;
}

Wikipediaにのっていたパターン

int IsLeapYear(int year)
{
    return year%4 == 0 && year%100 != 0 || year%400 == 0;
}

C言語を使う以上、パフォーマンスを気にするべきで後者のパターンを採用すべきだが、どちらかといえばわかりやすい前者のパターンのほうが好み。とはいえ今やJavaでもVBでも標準で備わっている機能だし、この判定に限っては後者を使うべきなのかもしれない。その下にコメントでも書いておけばいいと思う。
